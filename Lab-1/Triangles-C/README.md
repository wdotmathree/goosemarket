# Triangles

- triangle.c:
  - ./triangle n1 n2 n3 n4 n5 n6
  - Computes area of triangle formed by points (n1,n2), (n3,n4), and (n5,n6)
  - Has errors in the code

- driver.py:

    <code>./driver.py \<program\> \<testCases\></code>

  - a simple test-case driver that executes a set of testcases

- testCaseInput.txt
  - testcases of the form:
  
    <code>\<n\>, \<comma-separated command-line arguments\>, \<regexp\></code>
    
  - <code>n is the testcase number</code>
  - the program is run with the given <code>command-line arguments</code>
  - the output is matched against the <code>regexp</code>
