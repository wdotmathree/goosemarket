# Web-based Todo app

# Purpose
A web-based app that allows users to add, update, and delete tasks in a private todo list. Users can also view the next task, all tasks due today, and all tasks due tomorrow.

# Objective
This project aims to achieve basic functionality such that a locally hosted web server will allow the current user to interact with all aforementioned python functions from the web.

# Scope
The project will not include sharing todo lists with other users, customizing the todo website, viewing all tasks at once, or any other functionality not mentioned above.
